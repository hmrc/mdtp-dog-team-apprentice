Javascript
=========

Javascript is a fast-changing programming language. It's designed to run inside a web browser, and is frequently used to add interactivity and rich functionality to web pages.

Beginner
-------

* [Codecademy JS course](codecademy.com/tracks/javascript) - a long, but comprehensive course teaching basic programming principles using javascript. No project-based examples.

Node.js
-------

* [A basic HTTP web server](https://www.sitepoint.com/creating-a-http-server-in-node-js/)
