Prototype Count
====================

**Technology:** Any

**Difficulty**: Easy

**Duration**: 4-6 hours

**Relevant concepts**: requests, loops, schedule

**Prerequisites** Web Scraper

Goal
----
To track number of prototypes listed in the catalogue over time.

You should:
* Write request to get the catalogue page
* Count the number of services with the 'prototype' category
* Store the count 
* Schedule the request to run daily
* Calculate the number of prototypes created each week
* Graph prototype numbers growth over time

Stretch goals
------------

* Come up with a way modify the prototype creation jenkins job to notify when it has run successfully.

Possible approach
---------------
Similar to the webscraper. Don't forget you'll need VPN access to get the catalogue working.
