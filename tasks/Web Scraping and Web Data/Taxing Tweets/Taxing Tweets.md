Taxing Tweets
====================

**Technology:** Any

**Difficulty**: Medium

**Duration**: 16-32 hours

**Relevant concepts**: APIs, JSON, loops

**Prerequisites** Web Scraper project, API Module, JSON Module, Twitter account

Goal
----
Create a trending twitterfeed for HMRC and MDTP.

You should:
* Get Twitter API keys and install a relevant library
* Plug into Twitter’s search API
* Pull tweets related to HMRC and paying tax online
* Store the tweet data locally
* Display frequency of tweets 

Stretch goals
------------

* Allow a user to specify additional keywords for the search (e.g. VAT, 
* Plug into Twitter’s trend API and compare the frequency of tweets about tax and HMRC to other trending issues in the UK
* Perform rudimentary sentiment analysis on HMRC tweets

Possible approach
---------------
The web has a lot of tutorials and help for accessing Twitter’s APIs. This will give you a good start - http://socialmedia-class.org/twittertutorial.html
